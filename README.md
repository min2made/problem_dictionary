# 📚문제해결 백과사전
### 프로젝트 소개
중장년, 고령층이 IT 기기 사용 방법 숙지에 어려움을 겪어 자주 겪게 되는 사소한 문제들을 쉽게 해결할 수 있는 앱  
&nbsp; ●  다양한 문제 해결 방법이 기재되어 있는 백과사전  
&nbsp; ●  문제 해결 방법을 묻고 답 할 수 있는 커뮤니티  
&nbsp; ●  만약 해결하지 못한 문제가 있다면 질문 할 수 있는 AI 챗봇  


### 개발 환경
|분류|환경|
|------|---|
|프레임워크|Flutter, Firebase|
|언어|Dart|
|IDE|Android Studio|
